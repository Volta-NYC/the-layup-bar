# The LayUp Sports Bar
Official website for **The LayUp Sports Bar** — Brooklyn's sports bar located one block from Barclays Center.
 
---
 
## 🚀 Overview
 
This website serves as the official online presence for The LayUp Sports Bar, showcasing the menu, upcoming events, our story, hours, and location.
 
> 47 5th Ave, Brooklyn, NY 11217 · One block from Barclays Center
 
---
 
## 🛠 Tech Stack
 
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Hosted on Vercel
 
---
 
## 📂 Project Structure
 
- `src/app` → App Router (homepage, layout, globals)
- `src/app/pages` → Site pages (Menu, Events, Our Story, Contact)
- `src/lib/components` → Shared UI components (Navbar, Footer)
- `public` → Static assets
 
---
 
## 📄 Pages
 
| Page | Route |
|------|-------|
| Home | `/` |
| Menu | `/pages/menu` |
| Events | `/pages/events` |
| Our Story | `/pages/our-story` |
| Hours & Contact | `/pages/contact` |

## 🧑‍💻 Development
 
Install dependencies:
```bash
npm install
```
 
Run locally:
```bash
npm run dev
```
